dAVEBOx MIDI Jitter Analyzer

Captures MIDI note-on arrival timestamps via Web MIDI and compares inter-arrival jitter between two runs (A vs B). For verifying Phase 2 ROUTE_EXTERNAL latency improvement.
Setup
Open in Chrome/Edge over http://localhost (Web MIDI needs a secure context). Run python3 -m http.server 8080 in this folder, then open http://localhost:8080.
